4. AuthController ->
**AuthController için yazmanız gereken action'lar, uygulamanızın kimlik doğrulama gereksinimlerine bağlıdır.** 

**Genellikle, aşağıdaki action'lara ihtiyacınız olacaktır:**

**1. Login:** Kullanıcıların kimlik bilgilerini girerek oturum açmasına olanak tanır.

**2. Register:** Yeni kullanıcıların kaydolmasına olanak tanır.

**3. Logout:** Kullanıcıların oturumunu kapatmasına olanak tanır.

**4. ForgotPassword:** Kullanıcıların unutulan şifrelerini sıfırlamasına olanak tanır.

**5. ResetPassword:** Kullanıcıların şifrelerini sıfırlamasına olanak tanır.

**6. ChangePassword:** Kullanıcıların şifrelerini değiştirmesine olanak tanır.

**7. VerifyEmail:** Kullanıcıların e-posta adreslerini doğrulamasına olanak tanır.

**8. ExternalLogin:** Kullanıcıların Facebook, Google gibi harici sağlayıcılar aracılığıyla oturum açmasına olanak tanır.

**9. ConfirmEmail:** Kullanıcıların e-posta adreslerini onaylamasına olanak tanır.

**Ek action'lara ihtiyacınız olabilir:**

* Kullanıcı rollerini yönetmek
* Kullanıcı izinlerini yönetmek
* Kullanıcı profillerini yönetmek

**Action'ları yazarken:**

* **Güvenli kodlama uygulamalarını** takip ettiğinizden emin olun.
* **Hata işlemeyi** doğru şekilde uygulayın.
* **Kullanıcı dostu bir deneyim** sunmaya çalışın.

**Örnek:**

```csharp
public class AuthController : Controller
{
    // ...

    public ActionResult Login()
    {
        return View();
    }

    [HttpPost]
    public ActionResult Login(LoginViewModel model)
    {
        if (ModelState.IsValid)
        {
            // Kullanıcı kimlik bilgilerini doğrular.
            var user = _userManager.FindByNameAsync(model.Username).Result;
            if (user != null && _userManager.CheckPasswordAsync(user, model.Password).Result)
            {
                // Kullanıcıyı oturum açar.
                await _signInManager.SignInAsync(user, isPersistent: model.RememberMe);
                return RedirectToAction("Index", "Home");
            }

            ModelState.AddModelError("", "Kullanıcı adı veya parola hatalı.");
        }

        return View(model);
    }

    // ...
}
```
